You're being given a set of commands;
Each command is a string at the beginning of a line.
Some commands are followed by a set of whitespace-separated arguments.

1) crash - Crash the program, no arguments
2) sum <integer> <integer> - Print the sum of two numbers
3) wait <integer> - wait some time (in milliseconds)
4) end - return 0;
5) exit <integer> - exit with a specified exit code
6) continuity <integer> - quickly print specified number of lines.
	each line consists of its zero-based index and a random ~ 100-char aplhanumeric string, separated by a space
